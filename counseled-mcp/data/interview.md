# The family interview

Collect this conversationally — two or three questions at a time, not a form.
Explain *why* you're asking when it isn't obvious; families answer money
questions more honestly when they understand what the answer buys them.

## Student academics (drives merit likelihood)

| Item | Why it matters |
|---|---|
| Graduation year | Sets the timeline and how speculative the analysis is. Sophomores get directional answers; seniors get deadlines. |
| GPA — unweighted if possible | Compared against each school's average admitted GPA. Ask which scale if ambiguous. |
| SAT/ACT, taken or planned | The single strongest merit lever. Merit awards concentrate on students at or above a school's 75th percentile. If test-optional, note that many merit programs still require or favor scores — this changes advice. |
| Class rank / top-10% if known | Some auto-merit formulas (esp. Southern publics) key on rank. |
| Intended major(s) | Some merit programs are major-specific (engineering, music, nursing); some majors change which schools belong on the frontier at all. |
| Distinctive factors | Recruited athlete (athletic $ is a different system — flag it), National Merit (large auto awards at specific schools), arts portfolio, legacy, first-gen. |

## Preferences (constrains the frontier)

- Geography: regions they'd consider, max distance from home, urban/rural.
  Push gently on flexibility — the merit-generous list is geographically
  lumpy (heavier South and Midwest), and an inflexible geography constraint
  is often the single biggest cost driver.
- Size and vibe, only at coarse grain (small LAC / mid / big flagship).
- In-state options: always ask which state they live in. The in-state
  flagship (and its honors college) is the default financial safety and the
  benchmark everything else must beat.

## Money (drives everything else)

Ask plainly: **"What can you actually pay per year, all-in, including any
loans you're comfortable taking?"** Then:

- Clarify it's per-year, covering tuition, housing, food, books, travel.
- Distinguish comfortable number vs. stretch number if they offer a range —
  run the analysis at the comfortable number, note what the stretch unlocks.
- Household income bracket (rough): under $100k / $100–150k / $150–250k /
  over $250k. Purpose: a routing check, not a need calculation. Under ~$150k,
  meets-full-need schools (which give zero merit) may beat merit schools on
  actual price — if so, run both lenses and present both. Over ~$200k, need
  aid is unlikely almost everywhere and merit is the whole game; say so.
- 529/savings context only if volunteered — don't audit their finances.

## What NOT to do

- Don't ask for exact income, assets, or tax details. The skill's promise is
  that the family's financial details stay coarse and private.
- Don't ask twenty questions before delivering anything. After academics +
  budget + state you can run a first pass; refine from their reaction.
- Don't re-interview a returning family. If a tracker artifact or prior
  profile exists in the conversation or project, load it and ask only what
  changed ("new test score? budget still $X?").
